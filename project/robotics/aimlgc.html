Dermoscopic Lesion Imaging Classification

Jan 4, 2025
#Healthcare #MachineLearning #AI

This project focuses on building an automated system for diagnosing dermoscopic images of skin lesions using advanced AI techniques. The model addresses challenges in dermatology by providing accurate and timely classifications.

  • Built a multi-class classification model using the HAM10000 dataset to automate diagnosis from dermoscopic images.
  • Leveraged data augmentation techniques to increase the diversity of training samples and improve model generalization.
  • Utilized a CNN architecture to effectively extract features from dermoscopic images for accurate disease classification.
  • Worked in a 3-member team to fine-tune hyperparameters, achieving optimal classification results for all 7 categories.
